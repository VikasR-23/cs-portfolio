addEventListeners: Starting of the project we were introduced to addEventListeners. The main function for addEventListeners is so that you can add as specific effect or result of an input from the user such as a click,hover, etc. You can use these addEventListeners to place a certain function for one property or item, such as a object or button. They can be used to carry out a function for a whole set of objects. The image below displays a for loop using an array. If you look at the html code, you can see that each button has been assigned a class. Once so, we can go back to our javascirpt and write a getElementsByClassName with a variable assigned to it. The class name it will call will be the same as the one we assigned to all the buttons in the HTML file. We have now created an array. Finally the for loop uses this array to add a click property to all the buttons with only one line of code. But this isn't the only way to complete this task of adding a click property to the buttons. As I began building my calculator using an array wasn't my first choice. I originally started with a seperate onclick property for each single button in the calculator and connecting it to my javascript function. But later decided to use and array and addEventListener instead due to the convienience of having it all connected into one line of code.


getElementsByClassName/ById: Alongside the addEventListener we had to use getElementsByClassName. As shown before we used this to create the array we would call using the for loop. We created a variable and set that varibale to call the Id or Class we had set in the HTML file. And once we have done this we can use it multiple times to assign different properties and functions to it. Such as the click property we added. For the specifics of the for loop. To start off we assigned the value zero to a variable var i = 0. This is important because this is how the program will have to make a decision on it's own. Then we tell it that if the value of i is less then the array of values on the calculator, i < calcbutton.length;, add the click property to i. This is an example of telling the program to listen for an event that takes place on the webpage from the user, and the decision the program has to make based of the information we have told it.

innerHTML: The innerHTML property can be used in javascirpt to maniuplate the HTML page alongside with the outher DOM functions as I mentioned before such as getElementById/classname.
