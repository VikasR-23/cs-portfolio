HTML Basics
  • How to write a paragraph
    • In HTML to write a paragraph the tag used is the <p> and </p>, any text written between these tags will become a paragraph
  • How to insert headers
    • To insert headers and titles the <h1>and the </h1>are used, text written between these tags will become the headers
  • Inserting images
    • To insert a image you place the images url between the quotation marks in the the tag,<img src="">
  • Placing borders and changing fonts
    • To create borders and change fonts in HTML itself you can use the <style> and </style> tags to customize your text and webpage
CSS Basics
  • CSS file
    • If all the border designs, color, background, etc were done in the HTML file itself it would become hard to crowded which is why we use a seperate CSS file
  • How to change background colors without having to edit the html file
    • Insted of adding to our HTML file we can just write it in the css file by writing "body{ background-image: linear-gradient(#50b1e5, white);}" for example.
  • how to syle borders, text, width, etc in a seperate file
    • Just like the background image we can change the font size etc in the CSS file. After defining which tag we want to edit we can change various things, the padding,the border, the border color etc.
